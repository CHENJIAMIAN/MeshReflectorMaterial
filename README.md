# 提取自 `drei` 的 `Reflector` 和 `MeshReflectorMaterial`

本项目提供了独立于 React 和 Vue 的 `Reflector`、`MeshReflectorMaterial` 和 `ReflectorMesh`，实现了高质量的反射效果。
